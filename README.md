#  おすすめ漫画アプリ

ジャンルやキーワードから、おすすめの漫画を検索・表示できるWebアプリケーションです。

##  アプリ公開URL
https://manga-app-t6qn.onrender.com/

##  アプリの概要
ユーザーが「バトル」「ラブコメ」などのジャンルボタンをクリックもしくは、検索窓にキーワードを入力することで、
データベースに登録された漫画一覧から該当する作品を絞り込んで表示することができます。

##  使用技術
* **バックエンド:** Python (FastAPI)
* **フロントエンド:** HTML / CSS / JavaScript
* **データベース:** MySQL (TiDB Cloud)
* **インフラ・環境:** Docker / Render (デプロイ)
