#  おすすめ漫画アプリ

ジャンルやキーワードから、おすすめの漫画を検索・表示できるWebアプリケーションです。

##  アプリ公開URL
【ここにRenderのURLを貼り付けます：例 https://manga-app-xxxx.onrender.com】

##  アプリの概要
ユーザーが「バトル」「ラブコメ」などのジャンルボタンをクリックしたり、キーワードを入力することで、データベースに登録された漫画一覧から該当する作品を絞り込んで表示します。

##  使用技術
* **バックエンド:** Python (FastAPI)
* **フロントエンド:** HTML / CSS / JavaScript
* **データベース:** MySQL (TiDB Cloud)
* **インフラ・環境:** Docker / Render (デプロイ)

##  制作の背景・工夫した点
* **全体像の把握からスタート**
  プログラミング学習の一環として、まずはAIを活用しながら「フロントエンド・バックエンド・データベースがどう連携して動くのか」というWebアプリの全体像を体感することを目標に制作しました。
* **データベースの構築と連携**
  ローカル環境（Docker）での構築から始め、最終的にクラウドデータベース（TiDB Cloud）へ移行し、Renderを使って全世界にデプロイするまでのプロセスを完遂しました。途中で発生した環境構築やパスのエラー、SQLのテーブル構成のエラーなども、一つずつ原因を切り分けて解決しました。

##  ローカルでの動かし方（開発者向け）
1. このリポジトリをクローンします。
2. `pip install -r requirements.txt` で必要なライブラリをインストールします。
3. `uvicorn main:app --reload` でローカルサーバーを起動します。
