#  おすすめ漫画アプリ

ジャンルやキーワードから、おすすめの漫画を検索・表示できるWebアプリケーションです。

##  アプリ公開URL
https://manga-app-t6qn.onrender.com/

##  アプリの概要
ユーザーが「バトル」「ラブコメ」などのジャンルボタンをクリックもしくは、検索窓にキーワードを入力することで、
データベースに登録された漫画一覧から該当する作品を絞り込んで表示することができます。

##  使用技術
* **バックエンド:** Python (FastAPI)
* **フロントエンド:** HTML / CSS / JavaScript
* **データベース:** MySQL (TiDB Cloud)
* **インフラ・環境:** Docker / Render 

##  制作の背景・工夫した点
* **開発経緯**
Pythonを用いて、何か1つ成果物となるものを自身の手で開発したいと思ったため
* **全体像の把握**
 まずはAIを活用しながら「フロントエンド・バックエンド・データベース間ではどのような処理が行われているか」を自身で確認するために、
フロントエンド（HTML）のみで終わってしまうアプリではなく、サーバーでの処理やデータベースを使用することでどのようにアプリが動作するかを確かめました。
* **データベースの構築と連携**
